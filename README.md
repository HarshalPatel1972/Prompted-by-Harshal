# The Meta-Portfolio

Built by Harshal + An Army of Robots.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS 4
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Inter & Playfair Display

## Features
- **Glass-Bento Grid**: Responsive grid layout with glassmorphism.
- **Spotlight Effect**: Mouse-tracking radial glow.
- **3D Tilt Cards**: Interactive physics-based card tilting.
- **Floating Dock**: MacOS-style navigation with magnification.

## Getting Started

```bash
npm install
npm run dev
```
